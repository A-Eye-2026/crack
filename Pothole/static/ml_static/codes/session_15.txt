# 15강: 심층 신경망 (Deep Neural Network)

## 1. 개요
심층 신경망(Deep Neural Network, DNN)은 입력층과 출력층 사이에 여러 개의 은닉층(Hidden Layer)이 있는 인공 신경망입니다. 이번 장에서는 케라스(Keras) 패키지를 사용하여 다층 신경망을 구축하고, 활성화 함수와 옵티마이저의 변화에 따른 성능 차이를 확인합니다.

## 2. 데이터 준비 및 전처리
```python
import tensorflow as tf
from tensorflow import keras
from sklearn.model_selection import train_test_split

# 패션 MNIST 데이터 로드
(train_input, train_target), (test_input, test_target) = keras.datasets.fashion_mnist.load_data()

# 0~255 사이의 픽셀 값을 0~1 사이로 정규화
train_scaled = train_input / 255.0

# 훈련 세트와 검증 세트로 분할
train_scaled, val_scaled, train_target, val_target = train_test_split(
    train_scaled, train_target, test_size=0.2, random_state=42)
```

## 3. 심층 신경망 구축 (은닉층 추가)
은닉층에는 시그모이드(sigmoid)나 ReLU 활성화 함수를 주로 사용합니다.

### 방법 1: Sequential 클래스 생성 시 층 리스트 전달
```python
dense1 = keras.layers.Dense(100, activation='sigmoid', input_shape=(784,))
dense2 = keras.layers.Dense(10, activation='softmax')

model = keras.Sequential([dense1, dense2])
model.summary()
```

### 방법 2: Sequential 생성자 내에서 직접 정의 및 이름 부여
```python
model = keras.Sequential([
    keras.layers.Dense(100, activation='sigmoid', input_shape=(784,), name='hidden'),
    keras.layers.Dense(10, activation='softmax', name='output')
], name='Fashion_MNIST_Model')

model.summary()
```

## 4. Flatten 층과 ReLU 활성화 함수
`Flatten` 층은 입력 배열(28x28)을 1차원(784)으로 자동으로 펼쳐줍니다. `ReLU`는 은닉층에서 가장 선호되는 활성화 함수입니다.

```python
model = keras.Sequential()
model.add(keras.layers.Flatten(input_shape=(28, 28)))
model.add(keras.layers.Dense(100, activation='relu'))
model.add(keras.layers.Dense(10, activation='softmax'))

model.summary()
```

## 5. 모델 컴파일 및 훈련
적응적 학습률을 사용하는 `Adam` 옵티마이저를 사용해 봅니다.

```python
model.compile(optimizer='adam', loss='sparse_categorical_crossentropy', metrics=['accuracy'])

model.fit(train_scaled, train_target, epochs=5)
model.evaluate(val_scaled, val_target)
```

## 6. 다양한 옵티마이저 (Optimizer) 설정
훈련 도중 가중치를 최적화하는 다양한 알고리즘을 선택할 수 있습니다.

```python
# 1. SGD (성능 향상을 위한 모멘텀 추가)
sgd = keras.optimizers.SGD(learning_rate=0.1, momentum=0.9, nesterov=True)
model.compile(optimizer=sgd, loss='sparse_categorical_crossentropy', metrics=['accuracy'])

# 2. Adagrad (적응적 학습률)
adagrad = keras.optimizers.Adagrad()
model.compile(optimizer=adagrad, loss='sparse_categorical_crossentropy', metrics=['accuracy'])

# 3. RMSprop (적응적 학습률, 케라스 기본값)
rmsprop = keras.optimizers.RMSprop()
model.compile(optimizer=rmsprop, loss='sparse_categorical_crossentropy', metrics=['accuracy'])
```

## 7. 핵심 포인트 요약
- **은닉층**: 성능 향상을 위해 입력층과 출력층 사이에 추가하는 층입니다.
- **ReLU**: 시그모이드의 기울기 소실 현상을 해결한 현대적인 활성화 함수입니다.
- **Flatten**: 모델의 입력에서 데이터를 펼쳐주는 역할을 수행합니다.
- **Adam**: 모멘텀과 RMSprop의 장점을 합친 가장 인기 있는 옵티마이저입니다.
- **하이퍼파라미터**: 층의 개수, 뉴런 개수, 활성화 함수, 옵티마이저, 에포크 등 사람이 직접 설정해야 하는 변수입니다.
