은닉층 (Hidden Layer)의 필요성

단순한 인공 신경망은 입력층과 출력층만 존재합니다. 하지만 현실의 데이터는 비선형적인 복잡한 관계를 맺고 있는 경우가 많습니다.
은닉층은 입력 데이터를 추상적인 특징들로 변환하여 출력층으로 전달함으로써, 신경망이 훨씬 고차원적인 학습을 수행할 수 있게 합니다.

층을 추가하는 두 가지 방법

1. 리스트로 한 번에 정의

model = keras.Sequential([
    dense1, dense2
])

2. add() 메서드로 차례로 추가

model = keras.Sequential()
model.add(dense1)
model.add(dense2)
활성화 함수의 진화: Sigmoid vs ReLU

초기 딥러닝에서는 시그모이드 함수를 많이 사용했지만, 층이 깊어지면 출력값이 0이나 1에 가까워져 학습이 멈추는 기울기 소실(Vanishing Gradient) 문제가 발생했습니다.

[그림 15-1] 시그모이드와 ReLU 함수의 비교

ReLU (Rectified Linear Unit)

현대 딥러닝의 표준입니다. 양수일 때는 입력값을 그대로 통과시키고 음수는 0으로 차단하여 계산 속도가 빠르고 깊은 층에서도 학습이 잘 전달됩니다.

Sigmoid

출력이 0~1 사이로 제한되어 초창기 은닉층과 현재의 이진 분류 출력층에 사용되지만, 은닉층이 많아지면 성능이 저하되는 단점이 있습니다.

심층 신경망(DNN)의 학습 흐름

"어떻게, 언제 코드를 짜야 하나요?" 입문자가 가장 헷갈려 하는 딥러닝의 4대 핵심 질문을 정리했습니다.

1. 모델 설계

Sequential() 내에 층(Layer)을 쌓으며 뉴런 개수를 결정합니다.

2. 컴파일

compile()로 Optimizer 엔진과 손실 함수를 확정합니다.

3. 학습 (Fit)

fit()을 호출하여 실전에 돌입하며, 가중치가 갱신됩니다.

4. 평가 및 예측

evaluate(), predict()로 실전 성능을 냅니다.

데이터 전처리의 마법: Flatten 층

패션 MNIST 데이터는 28x28 크기의 2차원 배열입니다. 이를 밀집층에 넣으려면 한 줄로 길게 펼쳐야 합니다.

수동 변환 (과거)

넘파이의 reshape()를 사용하여 데이터를 미리 가공해야 했습니다.

train_scaled = train_input.reshape(-1, 28 * 28) / 255.0
Flatten 층 (현대)

모델 자체에 포함되어 입력 데이터를 자동으로 펼쳐줍니다. 가독성이 높아집니다.

model.add(keras.layers.Flatten(input_shape=(28, 28)))
모델 요약(Summary)과 파라미터
model = keras.Sequential([
    keras.layers.Flatten(input_shape=(28, 28)),
    keras.layers.Dense(100, activation='relu', name='hidden'),
    keras.layers.Dense(10, activation='softmax', name='output')
])
model.summary()
Layer (type) Output Shape Param #
flatten (Flatten) (None, 784) 0
hidden (Dense) (None, 100) 78,500 (784 * 100 + 100)
output (Dense) (None, 10) 1,010 (100 * 10 + 10)
옵티마이저(Optimizer) 정리

어떤 옵티마이저를 선택하느냐에 따라 학습의 품질과 속도가 결정됩니다. 현대 딥러닝에서는 **Adam**이 가장 무난한 기본값으로 널리 쓰입니다.

SGD

기본적인 경사 하강법
(속도 느림)

RMSprop

학습률 크기를 동적으로 조절
(Keras 기본값)

Adagrad

많이 변한 변수는 적게 갱신
(학습 멈춤 현상 존재)

Adam

모멘텀 + RMSprop의 장점 결합
(가장 널리 추천됨)
