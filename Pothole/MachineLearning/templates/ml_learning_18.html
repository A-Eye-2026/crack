실제 모델 분석기
핵심 파이썬 코드: 모델의 '속' 들여다보기

가중치(필터)와 특성 맵을 추출하는 실제 Keras 코드입니다. 이 코드를 통해 딥러닝 모델의 블랙박스를 해소할 수 있습니다.

# 1. 훈련된 모델 불러오기
model = keras.models.load_model('best-cnn-model.keras')

# 2. 첫 번째 합성곱 층의 '가중치(필터)'와 '절편' 추출
weights, biases = model.layers[0].get_weights()
# weights.shape -> (3, 3, 1, 32): 3x3 크기 필터가 32개
# 3. 중간 출력(특성 맵)을 얻기 위한 '함수형 모델' 정의
conv_layer = model.layers[0]
activation_model = keras.Model(inputs=model.input, outputs=conv_layer.output)

# 4. 테스트 이미지를 넣어 특성 맵 예측
feature_maps = activation_model.predict(test_image)

가중치 시각화의 목적

필터가 선명한 무늬를 띠면 학습이 잘 된 것이고, 노이즈가 심하면 학습이 덜 된 것을 의미합니다.

계층적 특징 추출 (TIP)

앞쪽 층은 '선/점'에, 뒤쪽 층은 '물체의 형태'에 반응하는 계층적 구조를 이해하는 핵심 도구입니다.

오늘의 핵심 데이터 & 기술적 기법 (TIP)
1. 데이터 전처리 (Resize) Fashion MNIST는 28x28입니다. CNN 입력 시 채널 차원(1)을 반드시 추가하여 (28, 28, 1)로 변환해야 합니다.
2. 특성 맵의 크기 Padding='same'을 쓰면 입력과 출력 크기가 같지만, 풀링 전후의 시각적 선명도가 달라지는 점에 유의하세요.
3. 모델 해석력 (XAI) AI가 왜 이런 결정을 했는지 증명하는 '설명 가능한 AI'의 첫걸음이 바로 시각화 기법입니다.
