                            훈련 과정 시각화: Loss & Accuracy
                        

                    

                    

                        

                            모델이 학습하면서 손실(Loss)이 줄어들고 정확도(Accuracy)가 올라가야 정상적인 학습입니다.
                            model.fit()이 반환하는 history 객체에 훈련 내역이 들어 있습니다.
                        


                        

                            
history = model.fit(train_scaled, train_target, epochs=5, verbose=0)
                            

                            

                            
# 학습 내역 시각화

                            
plt.plot(history.history['loss'])
                                # 손실률
                            

                            
plt.plot(history.history['accuracy'])
                                # 정확도
                            

                            
plt.show()

                        


                        

                            
                            핵심: history.history는 파이썬
                                딕셔너리로, 'loss'와 'accuracy' 키를 통해 에포크별 훈련 상태에 바로 접근할 수
                                있습니다.
                        

                    

                

            


            
            

                

                    

                        
검증 세트와 과대적합 탐지
                        

                        

                            과대적합(Overfitting)이란 훈련 데이터에만 너무 예민하게 적응하여, 모의고사보다 실전(테스트)에서 성적이 뚝 떨어지는 현상을 의미합니다.
                        


                        

                            

                                

                                    
정상 학습
                                    

                                    

                                        훈련 손실과 검증 손실이 함께 감소하는 구간입니다. 모델이 훈련 데이터의 지엽적인 특성이 아닌 일반적인 패턴을 잘 학습하고 있습니다.
                                    

                                

                                

                                    
과대적합 (Overfitting)

                                    

                                        훈련 손실은 계속 감소하지만 검증 손실이 다시 솟아오르기 시작하는 구간입니다. 억지로 기존 답안을 외우는 단계입니다.
                                    

                                

                            

                            

                                

                                    
# 검증 세트를 포함하여 학습 (val_loss
                                            기록용)

                                    
history = model.fit(train_scaled, train_target, epochs=20,

                                    

                                                            validation_data=(val_scaled,
                                        val_target))

                                    
# 훈련 손실 vs 검증 손실 비교 시각화

                                    
plt.plot(history.history['loss'])

                                    
plt.plot(history.history['val_loss'])

                                    
plt.legend(['train loss',
                                        'val loss'])
                                    

                                

                            

                        

                    

                

            


            
            

                

                    

                        
드롭아웃 (Dropout) 도입

                        

                            학습 과정에서 무작위로 은닉층의 뉴런 일부를 '꺼버리는(값=0)' 방법입니다. 특정 뉴런에만 정보를 과의존하는 것을 막아 과대적합을 억제합니다.
                        


                        

                            

                                

                                    

                                    
일반 신경망

                                    
모든 뉴런이 학습에 참여 (과로 위험)

                                

                            

                            

                                

                                    

                                    
Dropout 적용 중

                                    
일부 뉴런 휴식. 나머지가 역할을 분담하여 더 강건한 모델 형성

                                

                            

                            

                                

                                    

                                    
실전 예측 시점

                                    
모든 뉴런이 총동원되어 최종 평가를 진행 (케라스 자동 처리)

                                

                            

                        


                        

                            
model = keras.Sequential()

                            
model.add(keras.layers.Flatten(input_shape=(28, 28)))

                            
model.add(keras.layers.Dense(100,
                                activation='relu'))

                            
model.add(keras.layers.Dropout(0.3))
                                # 30%의 뉴런을 랜덤하게 비활성화
                            

                            
model.add(keras.layers.Dense(10,
                                activation='softmax'))

                        

                    

                

            


            
            

                

                    

                        
모델 파라미터 저장과 복원

                        

                            오랜 시간 훈련한 우수한 모델을 영구적으로 저장하고, 나중에 필요할 때 모델과 Weights를 그대로 가져올 수 있습니다.
                        


                        

                            

                                
가중치(Weights)만 저장

                                
모델의 아키텍처(층)는 코드로 짜고 파라미터 값만 불러옵니다.

                                

                                    model.save_weights('model-weights.h5')
                                

                                

                                    model.load_weights('model-weights.h5')
                                

                            

                            

                                
전체 구조(Structure)
                                    저장

                                
파일 하나에 빈 껍데기(아키텍처)와 알맹이(가중치)를 통째로 보관합니다.

                                

                                    model.save('model-whole.h5')
                                

                                

                                    model = keras.models.load_model('model-whole.h5')
                                

                            

                        

                    

                

            


            
            

                

                    

                        
콜백(Callback)과
                            조기종료(Early Stopping)

                        

                            모델의 손실 곡선을 지켜보며 언제 학습을 멈출지 인간이 대기할 필요가 없습니다. 최상의 에포크에서 자동으로 저장하고, 더 이상 가망이 없으면 알아서
                            훈련을 종료하는 고급 기술입니다.
                        


                        

                            

                                

                                    

                                        ModelCheckpoint

                                    
에포크마다 훈련 상태(val_loss)를 감시하다가, 예전 기록을 단 한 번이라도 경신(가장
                                        에러가 적음)하면 해당 찰나를 즉시 디스크에 저장(Save Best Only)합니다.

                                

                            

                            

                                

                                    

                                        EarlyStopping

                                    
과대적합 방지를 위해, 더 이상 기록이 좋아지지 않는 상태가 patience 횟수(예: 2에포크)만큼 연속되면 스스로 반복 루프를 강제 탈출합니다.
                                    

                                

                            

                        


                        

                            
# 체크포인트 - 검증 손실이 가장 훌륭할 때 무조건 파일 갱신
                            

                            
checkpoint_cb = keras.callbacks.ModelCheckpoint(

                            
    'best-model.h5', save_best_only=True)


                            
# 조기종료 - 최고 점수 도달 후 2번 더 해도 나아지지 않으면 훈련 중지.
                                    중지 후 최상의 폼으로 되돌리기(restore_best_weights).

                            
early_stopping_cb = keras.callbacks.EarlyStopping(

                            
    patience=2,
                                restore_best_weights=True)


                            
# 훈련 명령과 동시에 콜백 감시 카메라 장착

                            
history = model.fit(train_scaled, train_target, epochs=20, verbose=0,
                            

                            
    validation_data=(val_scaled, val_target),

                            
    callbacks=[checkpoint_cb, early_stopping_cb])
