딥러닝의 기초, 인공 신경망

인공 신경망(ANN)은 사람의 뇌 속에 있는 뉴런의 구조를 모방하여 만든 알고리즘입니다. 기존의 머신러닝 알고리즘으로 해결하기 어려웠던 복잡한 데이터(이미지, 음성 등)를 처리하는 데 탁월한 성능을 발휘합니다.

Fashion MNIST 데이터셋

이번 강의에서는 딥러닝 입문의 'Hello World'라 불리는 Fashion MNIST 데이터셋을 사용하여 10가지 종류의 패션 아이템을 분류해 보겠습니다.

데이터 로드 및 탐색

텐서플로의 케라스 패키지를 사용하면 유명한 데이터셋을 간편하게 내려받을 수 있습니다.

import tensorflow as tf
from tensorflow import keras
import numpy as np

# 데이터 로드
(train_input, train_target), (test_input, test_target) = keras.datasets.fashion_mnist.load_data()

# 데이터 크기 확인
print(train_input.shape, train_target.shape)
# (60000, 28, 28) (60000,) -> 28x28 픽셀 이미지 6만 개

0: 티셔츠, 1: 바지, 2: 스웨터, 3: 드레스, 4: 코트, 5: 샌달, 6: 셔츠, 7: 스니커즈, 8: 가방, 9: 앵클부츠

np.unique(train_target, return_counts=True) 결과, 각 클래스(0~9)마다 정확히 6,000개씩의 샘플이 들어있어 균형 잡힌 데이터셋임을 알 수 있습니다.
로지스틱 회귀 vs 인공 신경망

선형 방정식 (로지스틱 회귀)

각 클래스에 대해 독립적인 선형 방정식을 계산하여 결과를 도출합니다.

z_티셔츠 = w1*px1 + w2*px2 + ... + b

신경망 구조 (ANN)

선형 방정식을 '뉴런' 단위로 묶어 결합된 층(Layer)을 통과하며 더 복잡한 추론을 수행합니다.

입력층(데이터) -> 밀집층(가중치) -> 출력층(결과)
케라스로 신경망 구현하기

가장 기본이 되는 밀집층(Dense Layer)을 구성하고 모델을 학습시키는 과정입니다.

# 데이터 전처리: 0~1 사이로 스케일링 후 1차원으로 펼치기
train_scaled = train_input / 255.0
train_scaled = train_scaled.reshape(-1, 28*28)
# 밀집층 구성: 출력 뉴런 10개, softmax 활성화
dense = keras.layers.Dense(10, activation='softmax', input_shape=(784,))
# 모델 컴파일: 다중 분류 손실 함수
model.compile(loss='sparse_categorical_crossentropy', metrics='accuracy')
# 모델 학습
model.fit(train_scaled, train_target, epochs=5)

Scaling (정규화)

0~255의 픽셀값을 255.0으로 나누어 정규화하면 경사 하강법 학습이 훨씬 안정적이고 빠르게 진행됩니다.

Softmax

10개의 출력값을 확률의 꼴(총합 1)로 변환해 가장 가능성이 높은 클래스(옷 종류)를 골라줍니다.

Loss Function

종속 변수가 정수 형태의 다중 타깃일 때 sparse_categorical_crossentropy를 사용합니다.

훈련 결과 및 예측 분석

에포크 반복에 따른 손실값 감소와 정확도 향상 추이

소프트맥스 결과 분석

모델의 예측 결과(Output)는 결국 10개 뉴런의 각 출력값입니다. 가장 확신도가 높은 클래스를 모델의 최종 예측 결과로 선택합니다.
